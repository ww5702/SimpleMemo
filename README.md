테이블 뷰 구현 이론   
iOS에서 가장 많이 사용하는 패턴이 delegate 패턴   
1. 테이블 뷰 배치   
2. 프로토타입 셀 디자인, 셀 identifier 지정   
3. Data source,  delegate 연결   
4. Data source 구현   
5. Delegate 구현   
 
Delegate는 셀을 선택했을 때 넘어가는 값들   
즉 셀을 선택할 필요가 없는 앱이라면 구현할 필요가 없다는것.
